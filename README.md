# Habitus33 - Brain Optimizer & Reading Habit System

뇌 최적화와 독서 습관, 인지력 향상을 위한 신경과학 기반 SaaS. 단 16분, 몰입·집중·기억력을 완전히 새롭게 설계하세요.

## 프로젝트 소개

Habitus33는 신경과학 기반의 뇌 최적화 루틴, 독서 습관 트래킹, 인지 훈련(Brain Training) 등으로 사용자의 집중력·이해력·기억력·습관 형성을 지원하는 웹 서비스입니다.

- **TS 모드**: (구 TT모드) 예열 → 본독서 → 반추 → 결과의 4단계 집중 독서 사이클. 각 단계별로 UX/로직 분리, 몰입 루틴 제공
- **ZenGo 모드**: 다양한 인지 훈련 세션(기억력, 주의력, 논리력 등)과 피드백, 세션별 동적 라우팅 지원
- **33일 루틴**: 습관 형성 및 트래킹, 마일스톤 관리
- **독서 현황/통계/분석**: 대시보드, 분석, 리더보드, 여정, 공유 등 다양한 시각화와 피드백

## 운영자/클랜장(스터디장) 기능 및 전략

Habitus33는 단순 암기/학습 플랫폼을 넘어, 운영자(스터디장/동호회장/커뮤니티 리더)에게 다음과 같은 고도화된 권한과 동기부여 구조를 제공합니다.

### 1. 스터디 클랜(커뮤니티) 생성 및 운영
- 운영자는 자신만의 스터디 클랜(암기학습 그룹)을 1분만에 개설할 수 있습니다.
- 클랜별 커스텀 도메인/로고/소개/초대장, 운영자 프로필 강화, 브랜디드 콘텐츠(운영자만의 전용 암기게임/학습 템플릿) 개설 권한.
- 클랜별 공개/비공개/그룹 권한, 회원 초대 및 멤버 관리.

### 2. 암기학습 콘텐츠/게임 생성·관리
- 운영자 및 회원은 클랜 특성에 맞는 암기학습 콘텐츠(게임, 문제, 카드 등)를 자유롭게 생성·공유할 수 있습니다.
- 운영자는 브랜디드 콘텐츠(운영자 이름/로고/설명 포함)로 회원의 몰입과 소속감을 강화할 수 있습니다.

### 3. 성장/성과 시각화 및 리더보드
- 클랜별/운영자별 성장 리포트, 활동 데이터, 랭킹, 마일스톤, 멤버별 성과 시각화 제공.
- 리더보드, 랭킹, 사회적 증거(성공사례, 인증샷, 후기) 자동 노출 및 SNS/외부 공유 기능.

### 4. 공식 인증/특전 및 FOMO 구조
- 선착순 1000개 클랜 무료 개설(실제는 더 많이 허용, FOMO 유발), 1기 설립자 한정 공식 인증/배지/미디어 노출/운영자 전용 커뮤니티 제공.
- 운영자/클랜장만의 성장 리포트, 회원 초대장, 공식 인증서, 뉴스레터/미디어 노출 등 특전.

### 5. 온보딩/자동화/실행장벽 최소화
- 1분 온보딩, 템플릿/자동화(운영자 전용 가이드, 콘텐츠 샘플, 자동 성장 리포트)
- 운영자 전용 지원(FAQ, 1:1 컨설팅, 운영자 커뮤니티)
- 클랜 개설→회원 초대→콘텐츠 생성→성과 시각화→외부 공유→공식 인증까지 자동화 플로우

### 6. 심리적 기제/정체성 전환 설계
- 운영자/클랜장에게 '교육 설계자', '커뮤니티 리더'로서의 정체성, 브랜딩, 성장/성과 시각화, 사회적 증거, 공식 인증/특전, 즉시 실행/자동화, 외부 노출 등 강력한 참여동기 제공
- 리더십, 소속감, 인정욕구, 희소성, 성장, 사회적 비교, 브랜딩 등 심리적 기제를 다층적으로 자극

## ZenGo 모드: 기본 · 마이버스 · 오리지널

Habitus33의 ZenGo는 세 가지 모드로 제공됩니다.

### 1. 젠고 (기본 모드)
- 모든 사용자가 즉시 플레이할 수 있는 표준 인지훈련 모드입니다.
- 바둑 기반 기억력/주의력/논리력/반응속도 게임, 점수/피드백/랭킹/배지 등 공통 경험 제공.
- 예시: 3x3/5x5/7x7 바둑판에서 단어·문장·위치 기억, 제한시간 내 순서대로 클릭, 점수/반응속도/정확도 기반 랭킹.

### 2. 젠고 마이버스 (사용자 DIY)
- 사용자(또는 운영자/클랜장)가 직접 맞춤형 암기게임(문제, 카드, 시나리오 등)을 생성·관리·공유할 수 있는 DIY 모드입니다.
- 각 클랜/스터디/동호회별로 특화된 콘텐츠, 브랜디드 게임, 회원별 맞춤 학습이 가능합니다.
- 생성된 게임은 클랜 내 공유, 외부 초대, SNS 확산, 멤버별 성과 추적, 운영자 브랜딩 등 다양한 방식으로 활용됩니다.
- 예시: 특정 시험/과목/테마에 맞춘 암기게임, 운영자 이름/로고/설명 포함, 회원별 성장 데이터 시각화, 클랜별 리더보드.

### 3. 젠고 오리지널 (브랜디드 프리미엄 콘텐츠)
- 공식/전문가/브랜드가 직접 설계한 프리미엄 인지훈련 콘텐츠를 제공합니다.
- 검증된 커리큘럼, 고급 피드백, 프리미엄 배지, 공식 인증, 미디어/뉴스레터 노출 등 차별화된 경험.
- 예시: 유명 강사/기관/브랜드가 설계한 스페셜 암기게임, 한정판 콘텐츠, 공식 인증서 발급, 프리미엄 리더보드.

> 젠고(기본)는 표준/즉시, 마이버스는 DIY/맞춤, 오리지널은 공식/브랜디드 프리미엄으로 차별화됩니다.

## API 문서

API 문서는 [여기](docs/api.md)에서 확인하실 수 있습니다.

### 주요 API 엔드포인트

#### 도서 관리
- `GET /api/books` - 도서 목록 조회
- `GET /api/books/:id` - 도서 상세 정보 조회
- `POST /api/books` - 새 도서 등록
- `PUT /api/books/:id` - 도서 정보 수정
- `PUT /api/books/:id/progress` - 도서 진행상황 업데이트
- `DELETE /api/books/:id` - 도서 삭제

#### 독서 세션
- `POST /api/reading-sessions` - 새 독서 세션 시작
- `PUT /api/reading-sessions/:id` - 독서 세션 업데이트
- `GET /api/reading-sessions/stats` - 독서 통계 조회
