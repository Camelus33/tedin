# 핵심 가치
<!-- Screenshot placeholder -->
![bg opacity:0.06](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1600' height='900'><rect x='80' y='560' width='1440' height='260' rx='16' fill='%23ffffff' fill-opacity='0.06' stroke='%23a78bfa' stroke-dasharray='10,8' stroke-opacity='0.5' stroke-width='3'/><text x='150' y='700' fill='%23a78bfa' fill-opacity='0.8' font-size='28' font-family='Arial'>Screenshot placeholder</text></svg>)
