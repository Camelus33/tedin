## 🎯 개요

- **구체성 (25점)**: 실제 데이터와 수치를 포함한 현실적 시나리오

- **투자자별 차별화 (25점)**: 각 투자자의 고유한 철학과 스타일을 정확히 반영

- 영업이익: 31억 달러 (마진 26.1%, 전년 대비 0.3%p 개선)
- 자유현금흐름: 28억 달러 (전년 대비 12% 증가)
- 배당: 주당 0.46달러 (배당수익률 3.1%, 61년 연속 배당 증가)
