# AI 학습 진단사 - 수험생 특화 메모카드 검색 시스템 PRD

## 프로젝트 개요
기존 온톨로지 커맨드를 전면 개편하여, 수험생의 메모카드 데이터를 기반으로 한 AI 학습 진단사를 구축합니다.

## 비즈니스 목표
- 수험생의 학습 효율성 300% 향상
- 개인화된 AI 학습 코치 제공
- 실시간 학습 진단 및 최적화
- 시험 성공률 50% 향상

## 기술적 제약사항
- GPU 없음 (CPU 기반 처리만 가능)
- Render 백엔드 (서버리스 환경)
- MongoDB 데이터베이스
- Vercel 프론트엔드
- 최소한의 추가 인프라 비용

## 핵심 기능 요구사항

### 1. 고급 검색 시스템
- MongoDB Atlas Vector Search 통합
- OpenAI Embeddings API 활용
- 하이브리드 검색 (키워드 + 벡터)
- 수험생 특화 검색 쿼리
- 실시간 검색 (100ms 이내)

### 2. AI 학습 진단사
- 심층 학습 패턴 분석
- 예측형 학습 계획 생성
- 지식 연결 최적화
- 실시간 학습 최적화
- 취약점 자동 진단

### 3. 시간대별 학습 분석
- 5개 시간대별 메모카드 특징 분석
- 시간대별 효율성 분석
- 시간대별 학습 유형 최적화
- 시간대별 진도 추적

### 4. 시험 성공 예측
- AI 기반 성적 예측
- 개인화된 학습 전략
- 메타인지 학습 코치
- 적응형 학습 경로

### 5. 스마트 쿼리 제안
- 자주 사용하는 쿼리 패턴 학습
- 컨텍스트 기반 검색 제안
- 퍼플렉시티 스타일 채팅창 하단 제안

## 기술 스택
- 백엔드: Node.js + Express (Render)
- 데이터베이스: MongoDB Atlas (Vector Search)
- AI: OpenAI API (Embeddings + GPT)
- 프론트엔드: Next.js (Vercel)
- 검색: MongoDB Atlas Vector Search
- 캐싱: Redis (선택적)

## 성능 요구사항
- 검색 응답 시간: < 100ms
- AI 분석 응답 시간: < 3초
- 동시 사용자: 100명
- 데이터 처리량: 1000 요청/분

## 보안 요구사항
- JWT 인증
- 사용자 데이터 격리
- API 키 보안
- 입력 데이터 검증

## 확장성 요구사항
- 모듈화된 아키텍처
- 마이크로서비스 준비
- 다국어 지원 준비
- API 서비스화 준비
